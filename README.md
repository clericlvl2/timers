# Timers!

A simple countdown timer application built with React, TypeScript, and Effector.

https://clericlvl2.github.io/timers/

## Features

- Create multiple countdown timers
- Start, pause, resume and remove timers
- Real-time countdown display
- Persistent state across browser sessions
- UI separated from business logic using Effector

## Technologies

- React 19
- TypeScript
- Effector
- Ant Design
- Tailwind CSS 4
- Vite

## Installation

1. Clone the repository:

```bash
git clone https://github.com/clericlvl2/timers.git
```

2. Navigate to the project folder:

```bash
cd timers
```

3. Install dependencies and start the development server:

```bash
npm install
npm run dev
```
